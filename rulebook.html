KDS RuleBook 🚀GO! 사용 방법

업무 중 “이럴 땐 어떤 규정을 봐야 하지?” 하고 고민된 적 있으셨나요?
이제는 더 이상 규정 찾느라 헤매지 마세요.

KDS RuleBook 🚀GO!는
누구나 손쉽게 회사의 주요 규정들을 찾아볼 수 있도록 만든 규정 바로가기 대시보드입니다.

📌 주요 기능
1. 분야별 규정 모아보기
“조직, 복무, 사업관리, 사내 시스템” 등
각 규정이 유형별로 보기 쉽게 정리되어 있어요. 원하는 규정을 찾아 클릭만 하면 됩니다!

2. 키워드 검색 지원
①정확한 규정 이름이 생각나지 않는다면?
②또는 특정 내용이 알고 싶은데 어느 규정에 들어있는지 모르겠다면?
떠오르는 키워드만 입력해보세요.
예를 들어, ‘경조’라고 입력하면 ‘경조휴가’가 포함된 취업규칙을 보여줍니다!

🔍 사용 방법
1. 상단 검색창에 키워드를 입력합니다. (예: “출장”, “휴가”, “정보보안”)
2. 관련된 규정 목록이 자동으로 필터링됩니다.
3. 규정 이름을 클릭하면 바로 내용을 확인할 수 있습니다.

✔️ 규정은 계속 업데이트됩니다.
사용 중 개선 아이디어가 있다면 기획관리실로 알려주세요!
